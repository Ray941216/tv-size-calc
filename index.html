詳細算法

1. 以水平視野角推回螢幕寬度

設觀看距離為 d、目標水平視野角為 θ，則螢幕寬度：

width = 2 × d × tan(θ / 2)

2. 16:9 寬高與斜對角換算

16:9 螢幕的寬度係數為 16 / √(16² + 9²)，高度係數為 9 / √(16² + 9²)。

diagonal = width ÷ (16 / √337)

height = diagonal × (9 / √337)

3. 以「幾倍畫面高度」推回斜對角

部分規範使用畫面高度 H，例如 d = 3.1H 或 d = 1.5H。換成 16:9 斜對角：

diagonal = d ÷ (heightMultiple × 9 / √337)

4. 21:9 / 2.39:1 電影黑邊與有效高度

在 16:9 電視上播放 2.39:1 電影時，畫面通常吃滿寬度；有效畫面高度為：

activeHeight = screenWidth ÷ 2.39

因為 16:9 高度是 screenWidth ÷ 1.777…，所以 2.39:1 內容的有效高度約為 16:9 滿版高度的 1.777 / 2.39 ≈ 74.4%。這就是為什麼電影優先時，通常可以接受更大的電視。
